## 👋 关于我

我是一名计算机科学与技术本科生，关注 **AI Native 应用开发、Web 全栈、Serverless 架构与 ROS/Linux 机器人系统**。

我习惯使用 **Claude Code / Codex** 等 AI 辅助开发工具，把需求拆解、代码实现、调试验证和部署流程串联成稳定的工程工作流。相比单点技术，我更关注如何把 AI 接口、前端体验、数据处理、部署排障和真实硬件联调组合成可运行、可展示、可复盘的工程成果。

- 🤖 **AI Native**：LLM API 接入、Prompt 映射、AI 文案生成、fallback 机制
- ⚡ **Serverless Web**：Cloudflare Pages / Workers / D1、轻量后端、快速部署
- 🚗 **Robotics**：ROS1、Cartographer、RViz、Linux、SSH 远程调试
- 🧪 **Engineering**：Git 协作、CI/CD、systemd、日志分析、部署排障
